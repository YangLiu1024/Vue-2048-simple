# vue-2048-simple

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Development

```bash
#install vue-cli
npm install -g @vue/cli @vue/cli-init

#create vue project
vue init webpack-simple vue-2048-simple
```
